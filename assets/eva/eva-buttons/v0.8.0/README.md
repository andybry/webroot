# Buttons
Generic buttons.

## Usage

    <div class="button button-a">Blue Button</div>
    <div class="button button-b">White Button</div>
    <div class="button button-c">Red Button</div>

    <div class="button-group">
      <div class="button button-a">Blue Button</div>
      <div class="button button-b">White Button</div>
    </div>

    <div class="button button-a button-slant button-next">
      Button
    </div>
